# Calculator
A gallery app.

## About
manage your pictures and videos

- rename button
- light and dark mode
- slideshow mode
- lots of customization: animations, slide duration, thumbnail size

## Framework
This app uses [Flutter](https://flutter.dev/) and [Dart](https://dart.dev/).
